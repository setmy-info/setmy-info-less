# setmy-info-less-experimental

Prototype and in-progress CSS for LESS framework developers. This package is a scratch space
for new ideas, CSS experiments, and features that are not yet stable enough to land in the
main library packages.

**Target audience:** Developers working on the setmy-info-less framework itself.
Not intended for use in production projects.

## Install

```shell
npm i setmy-info-less-experimental
```

## Dependency

Depends on [`setmy-info-less`](../setmy-info-less/README.md).

## Warning

Contents of this package may change or be removed without a major version bump. Do not
build production CSS on top of this package.

## Development

```shell
npm run build
npm run lint:less
npm run verify
```
