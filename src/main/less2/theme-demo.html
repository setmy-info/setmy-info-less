🚀 Getting Started

To use the enhanced setmy-info-less theme system in your project:

  1. Include the compiled CSS file
  2. Add the theme-switcher.js script
  3. Initialize with new ThemeSwitcher()
  4. Use CSS custom properties in your styles
  5. Add theme toggle buttons with data-theme-toggle attributes

The system maintains full backward compatibility with existing setmy-info-less projects while adding powerful new theming capabilities.
